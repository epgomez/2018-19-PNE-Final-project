Genomic information

Page dedicated to the acquisition of information about the human and other vertebrates genome

How to use:

  • /listSpecies/:limit : Retrieves a list of all the available species in the database
    • /:limit is an optional endpoint that indicates the maximum number of species to be shown

  • /karyotype/:specie : Returns a list of the names (usually numbers) of all the chromosomes of the selected specie

  • /chromosomeLenght/:specie/:chromo : Returns the Length of the chromosome named "chromo" of the given specie
